## Description

Простой Node.js (nest.js) сервер — часть тестового задания от компании Tinkoff.

Маршруты:

- `/temperature` - данные о температуре

- `/precipitation` - данные об осадках

- `/` - заглушка

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
